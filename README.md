# Frontier Protocol

Frontier Protocol is a work-in-progress expert-mode modpack for Minecraft 1.21.1 on NeoForge. It begins with approachable but manual survival and develops through dimension-based progression into increasingly capable automation.

## Development Status

The pack is unreleased and under active development. Existing worlds are not guaranteed to remain compatible. Balance targets a single player first while avoiding designs that unnecessarily prevent LAN or server play.

## Repository Contents

- `kubejs/` contains custom recipes, events, data, and assets.
- `config/` contains shipped mod configuration.
- `defaultconfigs/` contains defaults applied to newly created worlds.
- `mods/` contains local mod JARs, which are intentionally excluded from Git.
- `resourcepacks/` and `shaderpacks/` contain optional visual assets.
- `AGENTS.md` documents development practices and design constraints.

## Mod List

See [`MODLIST.csv`](MODLIST.csv) for the versioned dependency inventory. CurseForge is the primary source; exact Modrinth matches are recorded when available. Installation packaging has not yet been finalized.
